About Ahoron

Ahoron is a comprehensive web platform designed to streamline academic collaboration and campus life. Whether you're a student, instructor, or administrator, Ahoron gives you all the tools you need to communicate, organize, and manage activities in one centralized place.

Our Mission

To empower educational communities by providing an intuitive, real-time system for messaging, club management, event coordination, and administrative oversightâ€”all backed by secure authentication and a responsive design.

Key Features

  • Student & Instructor Dashboards: Personalized interfaces to track courses, messages, and club activities.
  • Real-Time Chat: Powered by Socket.IO for instant messaging, typing indicators, and seen-receipt notifications.
  • Club Management: Create, approve, and join student clubs with an admin approval workflow.
  • Secure Authentication: Role-based login/signup using Node.js, Express, and MySQL.
  • Admin Panel: Full oversight of users, clubs, events, and system settings from a single control center.

Technology Stack

Built with modern web technologies: Node.js, Express, MySQL, Socket.IO, and a responsive HTML/CSS/JavaScript frontend.
